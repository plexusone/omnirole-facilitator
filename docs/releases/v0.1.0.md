# Release Notes - v0.1.0

**Release Date:** 2026-07-05

## Overview

Initial release of the Facilitator role for omnichannel meeting facilitation. This role enables AI agents to prepare for meetings, facilitate discussions in real-time, and generate documentation afterward.

## Installation

```bash
go get github.com/plexusone/omnirole-facilitator@v0.1.0
```

Requires Go 1.26+ and omniskill v0.10.0+.

## Highlights

- **Omnichannel Facilitation** - Works across meetings (OmniMeet), chat (Slack), and phone (Twilio)
- **Three-Phase Workflow** - Prepare, facilitate, and wrap-up workflows for complete meeting lifecycle
- **Session Tracking** - Track action items, decisions, and discussion points in real-time

## Features

### FacilitatorRole

Implements the `role.Role` interface from omniskill with:

- System prompts for each meeting phase
- RoleSpec with responsibilities, behaviors, artifacts, and metrics
- Session management for tracking meeting state

### Workflows

| Workflow | Description |
|----------|-------------|
| `prepare` | Gather pre-reads, create agenda, brief participants |
| `facilitate` | Track action items, decisions, and discussions |
| `wrapup` | Generate notes, publish to Confluence, create follow-ups |

### Required Skills

| Skill | Purpose |
|-------|---------|
| `meeting` | Join and participate in meetings via OmniMeet |
| `google` | Access Google Docs, Sheets, and Slides |
| `confluence` | Publish meeting notes to Confluence |

### Optional Skills

| Skill | Purpose |
|-------|---------|
| `chat` | Slack, Discord, Teams via OmniChat |
| `voice` | Phone calls via Twilio/OmniVoice |
| `aha` | Create features and initiatives in Aha |
| `jira` | Create and link Jira issues |
| `github` | Reference GitHub PRs and issues |
| `gitlab` | Reference GitLab MRs and issues |

## Quick Start

```go
import (
    "context"
    "github.com/plexusone/omnirole-facilitator"
)

role := facilitator.New(facilitator.Config{
    DefaultConfluenceSpace: "TEAM",
    EnableTranscription:    true,
    EnableActionTracking:   true,
})

err := role.Init(ctx, map[string]skill.Skill{
    "meeting":    meetingSkill,
    "google":     googleSkill,
    "confluence": confluenceSkill,
})
```
